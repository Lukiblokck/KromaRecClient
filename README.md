<h1 align="center">KromaRec Client</h1>

<p align="center">
  Open-source client for KromaRec
</p>

<p align="center">
  A custom Rec Room-compatible client designed to connect with KromaRec servers and experiment with alternative networking and gameplay systems.
</p>

<hr>

<h3 align="center">Project Goal</h3>

<p align="center">
  The goal of KromaRec Client is to provide a lightweight and flexible client that can connect to KromaRec server builds while keeping compatibility with existing Rec Room concepts.
</p>

<p align="center">
  This project is experimental and actively evolving. Some features may be incomplete or unstable during development.
</p>

<hr>

<h3 align="center">Platform Support</h3>

<p align="center">
  Currently focused on Windows development.
</p>

<p align="center">
  Linux support is planned, but will take time due to compatibility layers, rendering challenges, and testing requirements.
</p>

<p align="center">
  The long-term goal is to make the client fully cross-platform.
</p>

<hr>

<h3 align="center">Status</h3>

<p align="center">
  Early development stage — core systems are still being built.
</p>
